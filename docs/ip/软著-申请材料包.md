## 4. 直接可用的“软件说明”文案（按你当前项目）

### 4.1 软件全称
`Mini Bob 实时翻译系统 V1.0`

### 4.2 软件简称
`Mini Bob`

### 4.3 开发目的
本软件用于在桌面环境中实现“划词即译”的高效翻译体验，通过全局快捷键触发、自动读取选中文本、多翻译服务并行输出与流式渲染，降低跨应用阅读时的翻译成本。

### 4.4 主要功能
1. 全局快捷键触发翻译与偏好设置。
2. 自动读取当前选中文本（支持辅助功能读取 + 自动复制回退路径）。
3. 多服务独立配置（每个服务拥有独立 URL、API Key、模型、超时）。
4. 多服务并行翻译与流式输出，结果按服务分卡片展示。
5. 气泡模式/面板模式双形态展示，支持钉住、拖动、自动消失。
6. 术语表配置与文本自动化预处理（换行处理、注释符去除等）。
7. 偏好设置侧边栏管理，配置即时生效。

### 4.5 技术特点
1. 采用 Electron + Node.js 架构，主进程与渲染进程解耦。
2. 选区读取采用多路径协同机制，提升不同应用场景下的成功率。
3. 服务路由采用基于时延与失败统计的动态排序策略。
4. 翻译结果支持流式增量渲染与状态驱动更新，减少等待感。
5. 气泡窗口采用固定宽度、动态高度策略，降低布局抖动。

### 4.6 运行环境
- 操作系统：macOS（当前重点支持）/ Windows / Linux（按功能差异运行）
- 运行时：Node.js、Electron
- 网络：可访问所配置的大模型翻译服务接口
